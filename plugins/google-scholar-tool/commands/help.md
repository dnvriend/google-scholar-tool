Display help information for google-scholar-tool CLI commands.

## Usage

```bash
# Show general help
google-scholar-tool --help

# Show command-specific help
google-scholar-tool COMMAND --help

# Show version
google-scholar-tool --version
```

## Arguments

- `COMMAND` (optional): Specific command to get help for
- `--help` / `-h`: Show help information
- `--version` / `-v`: Show version information

## Examples

```bash
# General help
google-scholar-tool --help

# Command help
google-scholar-tool search --help

# Version information
google-scholar-tool --version
```

## Output

Displays usage information, available commands, and options.
